# みんなと自炊
## 🔗[みんなと自炊](https://everyone-jisui.vercel.app/)
## <img width="1918" height="989" alt="ogp" src="https://github.com/user-attachments/assets/79f6dcf4-9f27-4548-bfec-f730a64cd582" />


「みんなと自炊」は、自炊をきっかけにみんなと料理を楽しめるサービスです。


# 💡 このサービスを作った理由  
初めての一人暮らしをきっかけに、自炊をしてみました。
自炊をきっかけに料理が楽しく感じました。
今後、一人暮らしする上で自炊は欠かせないと思いますので、
他のユーザーの方達と共有しながら楽しく自炊(料理)ができるサービスを制作しました。

# 👥 想定ユーザー  
- 自炊をしている人
- 他のユーザーの自炊を参考に料理を始めたいかた
